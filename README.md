# Jayant Bakolia CPD Portfolio Website

A static GitHub Pages-ready CPD portfolio for the AERO62520 Robotic Systems Design Project.

## What is included

- `index.html` — homepage with bio, objectives, research interests, skills, links, and featured projects.
- `projects.html` — detailed project page with goals, contributions, media slots, and evidence links.
- `cpd.html` — feedback response, UK-SPEC evidence map, and three-part CPD plan.
- `cv.html` — web CV page with links to LinkedIn, GitHub, and CV download placeholder.
- `assets/css/styles.css` — responsive modern styling.
- `assets/js/main.js` — mobile menu behaviour.
- `assets/img/` — original placeholder SVGs; replace with your own screenshots/photos.
- `assets/videos/` — place demo videos here.
- `assets/docs/` — place CV PDF and project reports here.
- `.nojekyll` — disables Jekyll processing for simple static deployment.

## Before submitting

1. Replace `assets/img/avatar-placeholder.svg` with your real headshot or avatar.
2. Add at least 2–3 real media items per key project: screenshots, diagrams, and videos.
3. Add your actual CV PDF to `assets/docs/Jayant_Bakolia_CV.pdf`.
4. Replace all placeholder evidence links with real GitHub repositories, reports, demo videos, and/or cloud links.
5. Check that GitHub, CV, and LinkedIn links all work.
6. Add acknowledgements for any media that you did not create yourself.

## GitHub Pages deployment

Recommended simple route:

1. Create a public GitHub repository named `jayantbakolia.github.io`.
2. Upload these files to the repository root.
3. Go to repository **Settings → Pages**.
4. Choose **Deploy from a branch**.
5. Select branch `main` and folder `/root`.
6. Save and wait for GitHub Pages to publish.

Your likely URL will be:

`https://jayantbakolia.github.io/`
